O que é o Git?

O Git é um sistema de controle de versão distribuído amplamente usado para rastrear alterações em arquivos ao longo do tempo. Ele permite que várias pessoas colaborem em projetos de desenvolvimento de software de forma eficiente.

O que é o GitHub?

O GitHub é uma plataforma de hospedagem de código-fonte baseada na web que utiliza o Git para controle de versão. Ele oferece recursos para colaboração, gerenciamento de projetos e compartilhamento de código, tornando-o popular entre desenvolvedores e equipes de desenvolvimento.

Enviando arquivos para o GitHub

Para enviar arquivos criados, alterados ou excluídos do repositório local para o servidor do GitHub, siga os seguintes passos:

  1. Use o comando git add para preparar os arquivos que deseja enviar.
  2. Use o comando git commit para confirmar as mudanças localmente.
  3. Use o comando git push para enviar as mudanças para o servidor do GitHub.

Atualizando o repositório local

Para atualizar os arquivos do repositório local com possíveis alterações feitas no servidor do GitHub, você pode usar o seguinte comando:

git pull

Versionamento e Envio

Após criar esta página HTML, você pode iniciar o versionamento do projeto e enviá-lo para o GitHub. Certifique-se de inicializar um repositório Git localmente, vinculá-lo ao repositório do GitHub e, em seguida, use os comandos git add, git commit e git push para enviar a página para o GitHub.
